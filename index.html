📋 Petugas Sekolah Mengembalikan Rantang:
Pastikan semua rantang sudah dikumpulkan dari siswa dan diperiksa kondisinya sebelum dikembalikan ke SPPG.

Aplikasi MBG – Sekolah

Sistem informasi program Makan Bergizi Gratis untuk sekolah penerima. Memudahkan pencatatan absensi, pengajuan pesanan, dan pelaporan distribusi gizi kepada siswa.

🔌 Koneksi Supabase
📋 Cara Menghubungkan:
1. Buat project di supabase.com
2. Buka Settings → API
3. Salin Project URL & anon key
4. Paste di bawah, lalu klik Test Koneksi
5. ✅ Setelah terhubung: username & password bisa diubah dan tersinkron ke semua perangkat
